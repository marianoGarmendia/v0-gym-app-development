Así que lo construí una vez para no volver a hacerlo: un boilerplate con todo lo que necesita una aplicación real, y pensado para que la IA entienda el proyecto de principio a fin:

→ Auth con Google, GitHub, LinkedIn y email
→ Pagos con Stripe: suscripciones, webhooks, trials
→ Emails transaccionales con Resend y React Email
→ Blog con editor visual, categorías, tags y SEO
→ Newsletter y waitlist con sistema de referidos
→ i18n (español e inglés de serie, extensible)
→ Storage compatible con S3, Cloudflare R2 y Hetzner
→ Dark mode y 30+ componentes con shadcn/ui
→ Seguridad: CSP, rate limiting, GDPR
→ Charts con Recharts
→ Skills para que la IA sepa cómo trabaja el proyecto
→ Y más

No es solo un template con cosas preconfiguradas. Cada pieza está pensada para que la IA entienda la arquitectura, los patrones y las dependencias. Cuando le pides algo, genera código que encaja con el resto.
